# Инструкции по запуску локального сервера

## Проблема: npm не найден

Если вы видите ошибку "npm не найден", выполните одно из следующих:

### Вариант 1: Установить Node.js
1. Скачайте Node.js: https://nodejs.org/
2. Установите (включите опцию "Add to PATH")
3. Перезапустите терминал
4. Запустите: `npm run dev`

### Вариант 2: Использовать скрипт
Запустите PowerShell скрипт:
```powershell
.\start_local_server.ps1
```

### Вариант 3: Найти npm вручную
Если Node.js установлен, но не в PATH:
1. Найдите папку установки Node.js (обычно `C:\Program Files\nodejs\`)
2. Запустите:
```powershell
& "C:\Program Files\nodejs\npm.cmd" run dev
```

## После запуска

Сервер будет доступен по адресу: **http://localhost:5173**

Откройте этот адрес в браузере, чтобы увидеть обновленный дизайн.

## Альтернатива: Просмотр через build

Если не можете запустить dev сервер, можно собрать и посмотреть:

```powershell
npm run build
npm run preview
```

Это создаст production build и запустит preview сервер.
