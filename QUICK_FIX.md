# Быстрое решение проблемы с локальным сервером

## Проблема
npm не найден в PATH, Node.js не установлен или не добавлен в PATH.

## Решение 1: Установить Node.js (рекомендуется)

1. **Скачайте Node.js LTS**: https://nodejs.org/
2. **Установите** с опцией "Add to PATH"
3. **Перезапустите терминал/PowerShell**
4. **Запустите**:
   ```powershell
   npm install
   npm run dev
   ```

## Решение 2: Задеплоить на dev.ioperator.ai (быстрее)

Все изменения уже готовы в ветке `dev`. Можно сразу задеплоить:

1. **Закоммитить изменения** (если еще не закоммичены):
   ```powershell
   git add .
   git commit -m "Update design to match Gamma presentation"
   git push origin dev
   ```

2. **GitHub Actions автоматически задеплоит** на dev.ioperator.ai

3. **Проверить через 3-5 минут**: https://dev.ioperator.ai
